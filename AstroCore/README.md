Setup:

Note: All libraries need to be linked from the main CMakeLists.txt build config


1. Compile and add the RAYLIB library into the Astrocore/lib folder for the target/current platform
1. Compile and add the BOX2D library into the Astrocore/lib folder for the target/current platform [See More Info](https://box2d.org/)
2. Compile and install the STATIC version of the tmxlite library to the default directory for your system [More info here](https://github.com/fallahn/tmxlite)
